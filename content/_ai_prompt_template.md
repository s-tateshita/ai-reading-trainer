# AI自動生成用プロンプトテンプレート

## 概要
このファイルは、将来 AI（Claude 等）を使って問題コンテンツを自動生成する際に使用するプロンプト雛形です。

---

## 生成プロンプト（日本語）

```
あなたは家庭学習向け日本語読解問題の作問者です。
以下の条件に従って、content/_template.json の形式に沿った JSON を生成してください。

【日付】: {YYYY-MM-DD}
【週テーマ】: {週のテーマ}
【今日のテーマ】: {今日のテーマ}

【小１（grade1）の条件】
- 本文：150〜220字
- ひらがな中心の語彙（難しい漢字は使わない）
- 1段落30〜50字程度
- 具体的なエピソード形式
- 問題：事実確認・語彙・気持ち読み取り型を中心に5問
- 選択肢は4択、答えは本文に明示されている

【中１（grade7）の条件】
- 本文：400〜550字
- 説明文・論説文形式
- 難易度：中学入試〜中1教科書レベル
- 問題：事実確認・理由・語彙・推論・主題型を1問ずつ5問
- 選択肢は4択、論理的に正答が一つに絞れること

【出力形式】
- content/_template.json と完全に同じJSONスキーマに従うこと
- version: "1.0", reviewed_by_human: false, generation_model: "claude-opus-4-6"
- answerは0始まりのインデックス（0〜3）で指定

必ずJSON形式のみで回答してください。説明文は不要です。
```

---

## 自動反映手順（GitHub Actions連携想定）

1. 上記プロンプトをAIに送信し、JSON レスポンスを取得
2. `content/YYYY-MM-DD.json` として保存
3. JSON バリデーション（スキーマチェック）を実行
4. `main` ブランチにコミット・プッシュ
5. GitHub Actions が自動的にビルド・デプロイ

---

## バリデーションチェックリスト

- [ ] `date` が `YYYY-MM-DD` 形式
- [ ] `levels.grade1.passage` が 150〜220字
- [ ] `levels.grade7.passage` が 400〜550字
- [ ] 各レベルの `questions` が5問
- [ ] 各問の `choices` が4択
- [ ] `answer` が 0〜3 の整数
- [ ] `explanation` が空でない
